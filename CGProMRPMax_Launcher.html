How to open & install
1
Keep all 3 files in the same folder

Make sure CGProMRPMax_Launcher.html, CGProMRPMax_v32.html, CGProMRPMax_Phone_v32.html, server.js and Start_CGProMRPMax_Server.bat are all in the same folder (e.g. Desktop or Downloads).

2
Open ProMRP via browser URL (not file double-click)

Double-click Start_CGProMRPMax_Server.bat to start the server, then open Edge or Chrome and go to:

http://localhost:3000/CGProMRPMax_v32.html

This avoids all CORS restrictions and gives you full Firebase + PostgreSQL sync.

3
Create a Desktop shortcut (optional)

Right-click CGProMRPMax_Launcher.html → Send to → Desktop (create shortcut). Double-click it anytime to launch.

4
Pin to taskbar (optional)

Open in Chrome → click ⋮ menu → Save and share → Create shortcut → tick "Open as window" → Done. It will appear like a desktop app.

1
Use Chrome or Firefox on Mac (recommended)

Safari blocks some JavaScript features for local HTML files. Download Chrome for the best experience with ProMRP.

2
If you must use Safari — enable local file access

Open Safari → top menu Develop → tick "Disable Local File Restrictions". (If no Develop menu: Safari → Settings → Advanced → tick "Show Develop menu".)

3
Create a shortcut on Mac Desktop

Open file in Chrome → ⋮ menu → Save and share → Create shortcut. Or drag the file to your Dock from Finder.

4
Best option — host on a local server

Open Terminal and run: cd ~/Downloads && python3 -m http.server 8080 then open http://localhost:8080/ProMRP_Launcher.html in Safari. No restrictions!

1
Send the file to your phone

Open Safari or Chrome on your phone and go to:
cgpromrpmax.github.io/MRP/CGProMRPMax_Phone_v32.html
Or email yourself the link to open it easily.

2
Add to Home Screen (iPhone)

Open in Safari → tap the Share button (box with arrow) → scroll down → "Add to Home Screen" → Add. It'll appear as a ProMRP icon.

3
Add to Home Screen (Android)

Open in Chrome → tap ⋮ menu → "Add to Home screen" → Add. Works just like an app.

4
Enter Firebase URL in app settings

Open the phone app → tap ⚙️ Settings → paste your Firebase URL: https://promrp-415c3-default-rtdb.asia-southeast1.firebasedatabase.app

1
Make sure PostgreSQL is running

Open pgAdmin or run psql -U postgres to verify. Your database should be named promrp.

2
Start the ProMRP API server

Open Command Prompt, navigate to your ProMRP folder and run:

cd C:\CGProMRPMax && node server.js

You should see: ✅ Connected to PostgreSQL and 🚀 CGProMRPMax API running at http://localhost:3000/api

3
Open CGProMRPMax Desktop as usual

Click CGProMRPMax Desktop above. The app auto-detects the local server. When connected, data saves to both Firebase and PostgreSQL simultaneously.

4
Work offline (no internet)

With server.js running, ProMRP loads from PostgreSQL if Firebase is unreachable. You always have your latest data — even without internet.

💡
Pro tip: All data saves automatically to Firebase (Singapore server) AND PostgreSQL (local machine, when server.js is running). This means you can work even without internet — just keep the local server running. Use the 💾 Backup button occasionally for an extra JSON safety copy.
